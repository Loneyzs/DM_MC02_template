# 达妙科技 DM-MC-Board02 电机开发板引脚分配总结

## 1. 开发板概述
DM-MC-Board02 是一款面向机器人控制的高性能主控板。
* **核心芯片**: STM32H723VGT6 (Arm Cortex-M7, 550MHz)。
* **高速外部晶振**:24Mhz
* **电源输入**: 支持 12~24V (6S 供电)。
* **集成外设**: 3x CANFD, 2x RS485, 4x PWM, BMI088 IMU, 1.69寸 LCD 接口, QSPI Flash, DCMI 摄像头接口。

---

## 2. 完整引脚分配表

### 2.1 系统基础与电源控制
| 功能分类     | 信号名称            | 单片机引脚 (GPIO) | 硬件逻辑 / 备注                 |
| :----------- | :------------------ | :---------------- | :------------------------------ |
| **按键**     | 用户按键 (User Key) | **PA15**          | 外部上拉，按下接地 (低电平触发) |
|              | 复位按键 (MCU_RST)  | **NRST**          |                                 |
| **提示**     | 蜂鸣器 (Buzzer)     | **PB15**          | 定时器 TIM12_CH2，高电平触发    |
|              | WS2812 彩灯信号     | **PA07**          | SPI6_MOSI 信号控制              |
| **监测**     | 电源电压采集        | **PC04**          | ADC1_CH4_VBAT，分压电路监测     |
| **电源使能** | XT30 可控输出 1     | **PC14**          | 高电平使能 (Power_OUT1_EN)      |
|              | XT30 可控输出 2     | **PC13**          | 高电平使能 (Power_OUT2_EN)      |
|              | 对外 5V 电源使能    | **PC15**          | 高电平使能 (Power_5V_EN)        |

### 2.2 通讯接口
| 功能分类     | 信号名称            | 单片机引脚 (GPIO)      | 备注                       |
| :----------- | :------------------ | :--------------------- | :------------------------- |
| **CAN 总线** | CAN1 (RX/TX)        | **PD00 / PD01**        |                            |
|              | CAN2 (RX/TX)        | **PB05 / PB06**        |                            |
|              | CAN3 (RX/TX)        | **PD12 / PD13**        | 引出于 24Pin 扩展 IO       |
| **RS485**    | RS485-U2 (RX/TX/DE) | **PD06 / PD05 / PD04** | 转换芯片支持 10Mbps        |
|              | RS485-U3 (RX/TX/DE) | **PD09 / PD08 / PB14** |                            |
| **其他通讯** | SBUS 接收机信号     | **PD02**               | UART5_RX，带三极管反相电路 |

|              | USART1 (TX/RX)      | **PA09 / PA10**        | 常用作外部传感器或调试     |
|              | USART10 (TX/RX)     | **PE03 / PE02**        |                            |
